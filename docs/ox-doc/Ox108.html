Ox: Fixes and new features in version 1.08

New features and remarks, version 1.08

  • range() can now handle double arguments.
  • New functions: truncf() spline() nullspace() polygamma() format()
  • Increased accuracy of loggamma()

Fixed bugs in version 1.08

  • Assigning to an undeclared variable which has the same name as a function does not lead to a parse error.
  • Oxl.exe could have stack overflow on parsing very complex expressions.
