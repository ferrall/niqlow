virtual ImaiJainChing :: Candidate ( )
Default candidate new parameter vector.

This is in terms of the free parameters. Structural parameters are scaled and constrained transformations. This is different than IJC.

z ∼ N(0,IF.N)
C is the Cholesky decomposition of the candidate distribution variance matrix

 K

virtual ImaiJainChing :: K ( t )
Default Kernel in past parameters Ψ. Multidimensional Gaussian kernel across history of parameters, giving positive weight only to the ImainJainChing::Design[NClose] closest.
Parameters:
t iteration
Returns:
array {K*,near} 
Kh(Ψ-Ψ') = ∏s=0,&dots;Ψ.N(2π)-1/2exp[-0.5(ψs-ψ's)2]
near = indices of closest parameters (largest Kh)
K* = Kh[near]/∑Kh[near]
