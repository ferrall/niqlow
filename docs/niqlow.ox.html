 niqlow.ox

niqlow has several components that work together or separately
Click on the link to go to the documentation for that component
  • DDP: tools for designing and solving a dynamic programming problem
    • A simple search problem is solved by the code in examples/DDP/GetStarted.ox, and explained here
  • FiveO: tools for defining and maximizing an objective function or solving a non-linear system of equations.
    • The Rosenbrock function is coded and maximized by the code examples/FiveO/Rosenbrock.ox and explained here
  • If you ultimate goal is to estimate parameters of a dynamic program then your code will rely on both DDP and FiveO
  • CFMPI: tools to run and Ox code in parallel on a cluster using MPI library
