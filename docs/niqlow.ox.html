 niqlow.ox

niqlow has several components that work together or separately
Click on the link to go to the documentation for that component
Click on Examples link below to see how to run various code samples.
  • DDP: tools for designing and solving a dynamic programming problem
      • Among the help documents listed there:
    • A simple search problem is solved in examples/DDP/GetStarted.ox and explained in Get Started with DDP
    • You can also start with even simpler models of discrete choice as explained here
  • FiveO: tools for defining and maximizing an objective function or solving a non-linear system of equations
    • The famous Rosenbrock function is coded and maximized by the code examples/FiveO/Rosenbrock.ox and explained Get Started with FiveO
  • If your ultimate goal is to estimate parameters of a dynamic program then your code will rely on both DDP and FiveO
    • You might read my academic paper that discusses how to build a model from scratch and estimate it using niqlow.
    • That paper discusses some of the replications that are available in the Examples folder.
  • CFMPI: tools to run Ox code in parallel on a cluster using the MPI library
    • niqlow integrates MPI into estimation, so you can move your code seamlessly from a laptop to a supercomputer with no extra coding on your part.
    • Or, use the more basic interface routines to write your own MPI-ready applications.
